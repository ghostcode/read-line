# Read-Line

在网页上显示一条跟随鼠标移动的阅读辅助线，帮助锁定当前阅读行，减少视觉疲劳。

![版本](https://img.shields.io/badge/version-0.0.1-green) ![Manifest](https://img.shields.io/badge/manifest-V3-blue)

---

## 功能特性

- **阅读辅助线** — 一条横向线跟随鼠标移动，精准锁定当前阅读位置
- **三种线型** — 实线 / 虚线 / 波浪线，按需切换
- **10 种颜色** — 内置 10 种颜色可选，支持亮色和暗色页面
- **线宽可调** — 1px ~ 6px，自由调整
- **快捷键切换** — `Ctrl + Shift + L` 快速显示/隐藏
- **按网站隐藏** — 可向黑名单添加网站，在该网站自动隐藏辅助线
- **设置同步** — 通过 `chrome.storage.sync` 跨设备同步设置

## 安装

### 方式一：加载已解压的扩展（开发版）

1. 下载或克隆本仓库
2. 打开 Chrome，进入 `chrome://extensions/`
3. 开启右上角「**开发者模式**」
4. 点击「**加载已解压的扩展程序**」
5. 选择本项目的根目录

## 使用说明

安装后，打开任意网页即可看到绿色阅读线跟随鼠标移动。

点击浏览器工具栏中的 Read-Line 图标，打开设置弹窗：

| 设置项 | 说明 |
|--------|------|
| 启用阅读线 | 总开关，关闭后所有网站均不显示 |
| 线条颜色 | 点击色块切换颜色，实时生效 |
| 线条类型 | 实线 / 虚线 / 波浪线，点击切换 |
| 线宽 | 拖动滑块调整线条粗细（1~6px）|
| 禁止此网站显示阅读线 | 将当前网站加入/移出隐藏列表 |

### 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + Shift + L` | 切换当前标签页的阅读线显示/隐藏 |

> 可在 `chrome://extensions/shortcuts` 中自定义快捷键。

## 技术实现

- **Manifest V3** — 使用最新扩展规范，无后台页面（background.js 已移除）
- **纯原生 JS** — 无任何依赖、无构建工具，直接加载
- **SVG 渲染** — 阅读线使用 SVG 绘制，支持曲线（波浪线）
- **RAF 节流** — 用 `requestAnimationFrame` 节流 mousemove，性能最优
- **GPU 加速** — 用 `transform: translateY()` 代替 `top`，避免重排
- **最小权限** — 仅申请 `storage` 权限，不读取浏览历史

## 已知问题

- `chrome://` 内部页面和扩展页面无法注入 content script，在这些页面打开 popup 会显示「当前页面不支持」
- 页面未完全加载时打开 popup，可能暂时无法识别当前网站（关闭 popup 重新打开即可）
